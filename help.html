CQM Validator Help

    List of Methods:

  • /: Displays this page.
  • /help: Display this page.
  • /listSupportedMeasures: Returns list of supported CQM Measures
  • /validateMeasures[ccdaFile, measuresToTest]: Accepts a POST that includes the base64 encoded CCDA 'Transition of Care' document and the list of measures to test. It will return a JSON formatted report of the measures tested and if the patient would be included in the population for the numerator and denominator. If you use the keyword 'ALL' for the list of measures,it will test all of the supported measures.
